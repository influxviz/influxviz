About

InfluxViz is a set of client-side visualisation tools for an InfluxDB. The project is inspired by Influga and Grafana and is intended to power a similar dashboard designed specifically for influxdb.

Primarily intended for use by HyveDev, a home monitoring and automation solution, this software is MIT licensed and available for all users. Pull requests and help are welcome!
